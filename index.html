Movies

This was my first project as a part of the Udacity Full Stack Web Developer Nanodegree. I learned the basics of Python, including functions, conditionals, classes, and inheritence. And then used these concepts to create a simple movie listing page where users can select the poster art of a film to watch a trailer via YouTube.

Brain Drain

In this project I created a multi user blog (think primitive Facebook) where users can sign in and post blog posts as well as 'Like' and 'Comment' on other posts made on the blog. This site is hosted using Google's App Engine platform. I also implemented an authentication system for users to be able to register and sign in and then create blog posts!

ForSale

This project is essentially a basic Craigslist application, allowing users to post items for sale in a number of categories and arrange to buy from others. It implements basic CRUD database functionality thorugh an object relational mapper and a PostgreSQL database. Additionally, it implements 3rd party user authorization using Facebook and Goolge OAuth 2.0 standards.
